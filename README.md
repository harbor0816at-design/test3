# 业务管理系统（L1）

继续新增功能前，建议先阅读 `PROJECT_STATE.md` 与 `PROJECT_STATE.json`，当前实现结果和续开发入口已经保存。

## 模块层级

- L2：`客户报价模块`
- L2：`销售/收入分析模块`
- L2：`成本分析模块`
- L2：`经营情况分析模块`
- L2：`系统权限模块`
- L3：所有“数据录入/维护/上传”入口统一归类到 L3
- L2 导航：通过左侧侧边栏切换模块
- L3 展示：默认仅展示 1-2 行摘要，点击“展开/收起”查看详情

## 核心能力

- 客户报价查询：按 `客户 + SKU + 月份` 查询出货价
- 报价维护：调整 RRP 与红色区域字段（F-J）
- 报价数据录入：单条录入 + CSV/Excel 批量上传 + 修改后确认
- 销售/收入分析：按 `客户 + SKU + 月份` 录入销量并自动计算收入
- 销量批量上传：支持 CSV/Excel（.csv/.xlsx/.xls）
- 成本分析：按 `L1 + L2 + L3` 分类查看成本，支持手工录入、批量上传、发票识别上传、邮件/文字识别上传、自动识别分类
- 图片识别入口：成本录入区支持单张图片/发票/PDF 直接识别并回填表单字段
- 发票图片识别：统一调用 DeepSeek 进行图片/PDF 发票识别，并在信息不足时回退本地规则
- 成本上传识别：若已配置 DeepSeek API Key，上传时优先使用 DeepSeek 自动识别 L1/L2/L3；支持单张/多张发票图片、扫描 PDF 发票、邮件正文、文字说明识别，失败时自动回退本地规则或进入待确认
- 待确认修整：支持在成本模块中载入 `待确认` 记录，人工选择 L1/L2/L3 后保存修整结果
- 成本 AI 分析：支持基于当前筛选结果生成成本结构、未开票风险与优化建议
- 经营情况分析：覆盖 `2026-2028` 年，支持按 `月 / 季度 / 半年度 / 年度` 汇总销售收入、销售毛利、工厂补贴/返点、固定费用、变动费用、营业利润、净利润，并支持 AI 智能分析
- 结果展示：销售结果、收入结果、分客户/分型号/分月份汇总
- 毛利分析：基于 `FOB采购成本` 计算单台毛利、单台毛利率、总毛利，并提供分客户/分机型（品类）/分型号分析
- 简易 AI 分析：支持接入 DeepSeek API 生成分析建议
- 数据闭环：报价、销售、成本、经营补充均支持单条录入、编辑、批量导入后的“待确认 / 已确认”状态流转
- 页面体验：用户端优先展示查询与分析结果，录入入口折叠降低视觉噪音
- 汇总展示：分客户/分型号/分月份改为三行展示，每行支持展开/收起
- 看板风格：采用蓝色经营驾驶舱风格，保留白色录入卡片以提升数据维护效率

## 角色权限

1. 超级管理员：全部权限（报价维护/录入/管理、销量录入/上传/管理、毛利分析、账号管理）
2. 财务负责人：全部权限（与超级管理员一致）
3. 业务负责人：报价数据录入、报价数据管理、报价维护、销量数据录入/上传、销售结果与分析查看
4. 业务人员：查阅报价、查阅销售结果与分析、销量数据录入（不含报价维护与账号管理，不含毛利额/毛利率查看）

## 成本分析字段

- 必填：`costName`, `amount`, `orderCreatedAt`, `costOccurredAt`, `accrualStatus`
- 可选：`description`, `sourceText`, `l1`, `l2`, `l3`
- 自动匹配：系统会根据 `成本名称 / 说明 / 已传分类字段` 自动识别并归类到 `L1-L3`
- AI 识别：成本手工录入和批量上传均可复用 DeepSeek API Key 进行智能分类，支持中文 / 英文 / 德文内容识别，密钥仅保存在当前浏览器本地
- 权责发生时间：支持 `已发生已开票`、`已发生未开票`
- 人工修整：待确认成本可从下拉框或列表“修整”按钮载入，人工校正分类后再次入库

## 经营情况分析字段与映射

- 补充录入/上传字段：`month`, `factoryRebate`, `incomeTaxRate`, `remark`
- 销售收入：来自销售/收入分析模块 `salesRecords.revenue`
- 销售毛利：来自销售/收入分析模块 `salesRecords.grossProfit`
- 工厂补贴/返点：来自经营补充数据 `factoryRebate`
- 固定费用合计：来自成本分析模块中 `人力成本 / 行政办公 / 财务费用 / 其他费用`
- 变动费用合计：来自成本分析模块中 `差旅招待 / 物流费用 / 销售费用`
- 营业利润：`收入合计 - 固定费用合计 - 变动费用合计`
- 企业所得税：`营业利润 × incomeTaxRate`（亏损时按 0 处理）
- 净利润：`营业利润 - 企业所得税`
- 毛利率：`销售毛利 / 销售收入`
- 净利润率：`净利润 / 销售收入`

## 默认账号

- `superadmin / Admin@123`
- `finance / Finance@123`
- `saleslead / SalesLead@123`
- `sales / Sales@123`

## 报价公式

`出货价 = (RRP/(1+VAT)) * (1-Front Margin) * (1-DB点位-客户Margin-Service Fee) - STKbuffer`

## 批量上传字段

报价批量上传（CSV/Excel）字段：
- 必填：`customerName`, `sku`, `rrp1~rrp12(至少一项)`, `dbRate`, `customerMargin`, `serviceFee`, `stkBuffer`, `frontMargin`
- 可选：`category`, `vatRate`, `ura`

销量批量上传（CSV/Excel）字段：
- 必填：`customerName`, `sku`, `saleMonth(YYYY-MM)`, `quantity`
- 必填（毛利）：`unitCost`（FOB采购成本）
- 可选：`unitPrice`（留空则自动按报价计算）, `category`

## DeepSeek AI 分析

- 在“销售/收入分析模块”中输入 DeepSeek API Key 后点击“生成 AI 分析”
- 在“成本分析模块”中可复用同一 API Key，进行发票识别、费用分类与成本 AI 分析
- 在“经营情况分析模块”中可复用同一 API Key，进行经营质量与利润结构分析
- 接口：`https://api.deepseek.com/v1/chat/completions`
- 密钥仅保存在当前浏览器本地（localStorage）
