Quote Engine

报价公式自动化

出货价 = (RRP / (1 + VAT)) × (1 - Front Margin) × (1 - DB点位 - 客户Margin - Service Fee) - STKbuffer

可录入：客户、SKU、分月RRP、DB、Margin、Service Fee、STKbuffer、Front Margin、VAT、URA 自动化：分月出货价、销售单价回填、收入、毛利、经营分析
系统会按客户 + SKU + 月份自动取值，除核心点位外，其余结果统一自动计算。

L2 成本分析模块

L3 成本数据录入

录入成本名称、金额、订单生成时间、成本发生时间，系统自动识别并匹配 L1-L3 分类；支持修改与确认。

系统先用本地规则做快速预判；如已配置 DeepSeek，则可进一步结合发票图片、扫描 PDF、邮件/原始文字信息做中文、英文、德文的费用科目识别；保存后可进入确认流程。

L3 成本数据录入｜批量上传 CSV / Excel

上传后自动识别并匹配到对应 L1 / L2 / L3 分类。

支持三类上传：1）CSV / Excel 批量导入；2）单张或多张发票图片 / PDF 智能识别；3）邮件/文字文件识别。建议字段：costName、amount、orderCreatedAt、costOccurredAt、accrualStatus，可选字段：description、sourceText、l1、l2、l3。

上传时若存在有效 DeepSeek 密钥，系统会优先调用 AI 识别发票图片、扫描 PDF、邮件正文或文字说明；失败时自动回退到本地规则识别或待确认。

L3 待确认成本人工修整

对未识别或待确认的成本记录进行人工归类修整，修整后将进入正式 L1 / L2 / L3 分类。
